# @myinst/cli

CLI oficial do MyInst para login, listagem, pull e push do vault.

## Instalacao

```bash
npm install -g @myinst/cli
```

## Uso

```bash
myinst --help
myinst login
myinst list
myinst pull
myinst push
```

## Requisitos

- Node.js 22+
- acesso a um servidor MyInst
