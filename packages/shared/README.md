# @myinst/shared

Schemas, tipos e contratos compartilhados do ecossistema MyInst.

## Instalacao

```bash
npm install @myinst/shared
```

## O que este pacote expoe

- constantes de dominio
- schemas Zod usados pela API e clientes
- tipos TypeScript compartilhados

## Uso

```ts
import { createContentSchema } from '@myinst/shared';
```
